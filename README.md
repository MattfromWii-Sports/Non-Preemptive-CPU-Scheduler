# Non-Preemptive-CPU-Scheduler

CPU Scheduling for Non-preemptive algorithms

- FCFS
- SJF
- Priority
- Deadline
- MLQ

1. Solve for the Gantt chart while updating the attributes of the page objects.
   - Stores the gantt values in an array.
2. Renders the table based on the object attributes.
3. Renders the Gantt chart.
   - Uses the gantt array.
4. Profit!
